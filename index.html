Rodan; A workflow engine, developed by the Distributed Digital Music Archives Libraries (DDMAL) lab

Upload Resources

You can choose the type of the Resource you are adding or choose the default type (application/octet-stream)

Run Workflows

Managing Workflows is the main feature provided by Rodan. After creating a Workflow, add one or more Jobs and connect them to make a valid Workflow. Run the Workflow and wait for the status to change to FINISHED.

Open Source

Rodan is 100% free software, meaning that anyone can modify or distribute it. Help us improve and extend Rodan by forking our GitHub repository.
