Rodan is a web-based document recognition system; A workflow engine, developed by the Distributed Digital Music Archives Libraries (DDMAL) lab

Upload resources.

You can choose the type of the resource you are adding or choose the default type (application/octet-stream)

Run a workflow

Workflow is the main feature provided by Rodan. After creating a Workflow, add one or more jobs and connect them to make a valid Workflow. You can only connect an output of a job to input of another when they use the same resource type. Running a Workflow will result in a WorkflowRun which you can see its status.

Open Source

Rodan is 100% free software, meaning that anyone can modify or distribute it. Help us improve and extend Rodan by forking our GitHub repository.
