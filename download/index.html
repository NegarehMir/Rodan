1. Download the release package.

This package contains everything you need to set up Diva.js on the front-end.

Download diva-v{{site.data.release.version}}.tar.gz

The latest stable version is {{site.data.release.version}}. (Release Notes + Source Code)

2. Configure an image server.

Diva.js works using any IIIF-compatible image server package. We recommend IIPImage Server.

Follow our instructions to set one up.

3. Integrate into your existing website.

Get a basic setup running by following our instructions for configuring the front-end.

Then, check out our documentation on Integrating Diva into your own website.

4. (Optional) Contribute to Diva.js.

Fork or clone our GitHub Repository:

git clone git://github.com/DDMAL/diva.js.git

Send us a pull request!
